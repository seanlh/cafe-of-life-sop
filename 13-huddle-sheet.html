If undecided: write undecided, add follow-up, update once they choose.

Day 3

Setup check:

If missing: add it to post-shift follow-up before you leave.

SoftWave

If missing: call, document attempt, flag it so it does not get skipped.

Special / Other

If unsure: write the question and mark Waiting on Dr. / manager.
