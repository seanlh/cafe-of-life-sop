🌷 Import Payments into Cash Practice

  • Go to Cashpractice.com — bookmark should be in Chrome
  • Dashboard: Review all # Transactions > Payments
  • Click Review All # Transactions
  • Exports > Export ChiroTouch File
  • Download Transaction Report to Downloads folder
  • Open Cash Practice Transaction Importer
  • Select Next > Import Payments
  • Browse to today's Transaction Report in Downloads
  • Press Next again on the payment summary
  • Select REPORT and print the report
  • Finish, OK, close Importer, and return to CashPractice
  • Select the white box below Archive to check off successful payments in green
  • Archive payments when all successful payments are selected

🌷 Quick Rules

  • Pick up phone within first 2 rings
  • Always give TWO scheduling options
  • Charge BEFORE services rendered
  • Reply to all messages / texts same day
  • Initial and date every note you write (never archive messages in Review Wave)

🌷 Declined Payment Workflow

⚠ Do not ignore red items If Cash Practice shows a red / declined payment, check the patient ledger in CT, verify the amount, call the patient, document the attempt, and notify Dr. Jen / manager if needed.
