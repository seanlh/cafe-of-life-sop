Opening Procedures

Do all of this before the first patient arrives

Enter & Set Up

  • Alarm OFF — panel on LEFT side of entry
  • Clock in
  • Lights on
  • Left TV → Italy / Scenic / Travel / Relaxing (vibe only)
  • Other TVs → YouTube → Library → Playlists → ADJ Room Playlist
  • Diffuser: 3–4 drops · move AWAY from plant
  • AC → 72°F
  • Computers on · log into check-in station
  • X-Ray computer on
  • X-Ray machine on — turn on BOTH green buttons + Switch B + power supply panel
⚠ Do not water plants Dr. Jen and Aunt handle Wed & Sat. Ask first.

Communications

  • Check voicemail — write down every message
  • Return all calls / voicemails
  • Check office email — reply to all emails
  • Check Review Wave — reply to all texts
🌸 Note Do NOT archive messages. Reply same day.

🌷 Cash Practice

  • Open Cash Practice
  • Import payments into ChiroTouch
  • Verify ledger in CT — payments posted correctly
  • Verify balances match
🌸 If unsure Anything looks off → ask Dr. Jen.

🌷 Schedule Review

  • Open ChiroTouch — review today's full schedule
  • Pull charts for every patient · organize by appointment time
  • Prepare rooms · fresh face paper on every table
  • Queue SoftWave videos · set timers
  • Sweep floors (Tuesday · Thursday · Friday only)
